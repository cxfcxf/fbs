# Android TV 文件服务器 (FBS)

一个将您的 Android TV 变成文件服务器的应用程序，允许通过 Web 浏览器轻松管理文件。

[English](README.md) | 简体中文

## 功能特性

- **高密度网格界面**：优化的 10 列布局，在大屏幕上显示更多文件。
- **Web 管理界面**：全功能的浏览器界面，可从电脑或手机管理文件。
    - **上传**：直接将文件拖拽到当前查看的目录。
    - **删除**：远程删除文件和文件夹。
    - **导航**：独立的浏览历史和目录记忆功能。
- **电视控制**：
    - **优化导航**：智能的方向键支持。
    - **快捷操作**：在电视上**长按**确定键直接删除文件。
- **内置 HTTP 服务器**：稳定可靠的文件传输服务。

## 使用方法

### 电视界面

1.  **启动应用**：根据提示授予存储权限。
2.  **浏览文件**：
    - 使用方向键在 10 列网格中导航。
    - **点击（确定键）**打开文件或进入文件夹。
    - **长按（确定键）**在文件/文件夹上以删除它。
3.  **启动服务器**：
    - 导航到右上角的 **"Start"** 按钮。
    - 点击以开启/关闭服务器。
    - 屏幕上按钮旁会显示服务器 URL（例如 `http://192.168.1.x:8080`）。

### Web 界面

1.  确保您的电脑/手机与电视连接在同一个 Wi-Fi 网络。
2.  在浏览器中打开电视屏幕上显示的服务器 URL。
3.  **管理文件**：
    - **导航**：点击文件夹进行浏览。URL 会自动更新，因此您可以收藏特定文件夹。
    - **上传**：将文件拖拽到页面任意位置，或使用“选择文件”按钮。文件将上传到**当前**文件夹。
    - **删除**：点击任意项目旁边的垃圾桶图标 (🗑️) 即可删除。

## 开发

### 要求

- Android Studio Iguana 或更新版本
- JDK 21
- Android API 34+ SDK

### 构建

1.  克隆仓库。
2.  在 Android Studio 中打开项目。
3.  使用 Gradle 构建：
    ```bash
    ./gradlew assembleDebug
    ```
4.  通过 ADB 安装到设备：
    ```bash
    adb install -r app/build/outputs/apk/debug/fbs.apk
    ```
